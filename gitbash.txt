1. https://git-scm.com/downloads 에서 git 설치

2. git bash 터미널(명령 프롬프트)에서 아래와 같은 순서로 명령어 입력
   //사용자(adobe-illust) 확인
   가. git config --global user.name "사용자"            
   //사용자 이메일 주소 확인
   나. git config --global user.email "메일 주소"
   //원격 저장소 정보를 로컬로 복제        
   다. git clone https://github.com/사용자/저장소.git"   
   //상태 확인
   라. git status
3. 로컬 컴퓨터의 파일을 저장소로 업로드
   //스테이징 영역(가상 공간)에 추가
   가. git add 폴더/파일
   예) git add video/file-01.mp4
   // master에 커밋할 메시지 기록(변경 사항 메모, 버전관리)
   나. git commit -m "메시지"
   예) git commit -m "Create file-01.mp4"
   //참고: Permiision 오류 발생 시 제어판 > 사용자계정 > 자격증명관리자 >Windows 자격 증명 에서  git:https://github.com 제거 후 다시 로그인
   //원격 저장소로 업로드(푸시)
   다. git push origin master
