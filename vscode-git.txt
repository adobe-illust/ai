vscode에서 git 사용하기
1. https://github.com 로그인 후 저장소(reporitoy) 생성
2. vscode 에서 저장소 정보를 로컬로 복제
   가. vscode 명령팔레트(ctrl+shift+p): git clone
   나. 저장소 주소 입력: https://github.com/사용자/저장소.git
   다. 리포지토리 위치 선택: 로컬 컴퓨터의 폴더 지정
3. vscode 탐색기에서 로컬 폴더에 업로드할 새 파일 저장
4. vscode 소스 제어에서 커밋 메시지를 기록하고 커밋(ctrl+enter): 알림창이 나오면 "모든 변경 사항을 자동으로 스테이징하고 직접 저장" 예
5. vscode 소스 제어의 상단 우측 목록 메뉴에서 푸시(push): 로그인 정보 입력
6. 참고: 파일 수정 사항을 업로드 하는 경우는 vscode 소스 제어에서 커밋(commit), 풀(pull), 푸시(push) 순으로 실행한다.
